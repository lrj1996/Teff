This is the code on data analysis and figure generation for the manuscript titled Reconciling geographic variations in particle transfer efficiency through the ocean’s twilight zone.

TE_cluster.m is the script to do time series clustering.

Figure1-Figure3.m are scripts to generate figures in the manuscript.
To reproduce the figure, you need to download m_map from https://www-old.eoas.ubc.ca/~rich/map.html and add it to the work path.
